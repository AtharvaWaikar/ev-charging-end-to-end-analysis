================================================================================
‚ùå TROUBLESHOOTING
================================================================================

ISSUE: "FileNotFoundError: ev_charging_data.csv not found"
SOLUTION: Make sure CSV file is in SAME folder as app.py

ISSUE: "ModuleNotFoundError: No module named 'streamlit'"
SOLUTION: Run: pip install -r requirements.txt
         Or manually: pip install streamlit pandas numpy scikit-learn

ISSUE: Port 8501 already in use
SOLUTION: Run: streamlit run app.py --server.port 8502
         (Or use any other free port number)

ISSUE: App loads very slowly
SOLUTION: This is normal on first load (5-10 seconds)
         Subsequent loads are instant

ISSUE: No data appears
SOLUTION: Check CSV file has 50 rows (not corrupted)
         Make sure column names match exactly
